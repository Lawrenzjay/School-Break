SCHOOL BREAK

School Break is a game inspired by the infamous
Google Offline Game the Dinosaur and the Super Mario Game.
This game showcase a lot of new yet familiar features
that will keep you hooked and entertained.
This is a game that is made for everyone,
a game catered to all ages and genders
with the main goal of providing a fun, joyful,
and an entertaining gaming experience.
